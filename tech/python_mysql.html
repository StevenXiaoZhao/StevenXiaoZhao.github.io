Python 3.x 连接 mysql 有许多方法，这里说一下怎么用 connector-python 解决数据库连接问题。前提是你已经安装了mysql和python。这时，只需要下载安装即可，具体按照官方网站的步骤解决即可： http://dev.mysql.com/doc/connector-python/en/connector-python-installation.html

我们这里想解决的问题是如果按照这些步骤安装以后还是没有mysql.connector,这时说明我们安装了不止一个版本的python。最大的可能就是默认的python是python2，但是目前我们需要python3，既然这样，那我们只需要将安装在python2下面的package copy到 python3下面即可。

查看package的位置可以参考：http://dev.mysql.com/doc/connector-python/en/connector-python-verification.html
